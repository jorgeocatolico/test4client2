Entities

Understanding Entities

The entities screen

Selecting an entity with a single click, reveals all associated processes, properties and attributes on the right hand panel as shown below. Entities are viewed in read-only mode by default.

Search for an entity

This functionality is used when there are a large number of entities in the tree and provides an alternative to scrolling through tree structures.

  1. From the top of the tree, type the entity code or entity description into the search field.

  2. The search results will be displayed below the search box.

  3. Select the required entity.

 

 Navigating between entities

pControl entities have links to other entities via attributes. Attributes that are entities display as hyperlinked text as shown below in the read-only view. Select the link to open the entity in a new tab.

 

Creating New Entities

Create a new entity

To create new entities within the library tree:

  1. Select the required location in the library tree, right click and select 'Add'.

  2. Select a template group and template type from the drop down menus.

    Note
    Entity templates are used as the basis for the creation of a new entity. They define the appearance, and valid process types applicable to an entity. Entity templates are configured in System Administration.
  3. Select <Continue> to continue creating a new entity, or <Cancel> to exit without saving.

  4. If continuing, a blank entity will open.

    Enter a unique entity code and description; select the 'Active' flag; and populate any other attributes required for processing.

  5. Select <Save> to save the new entity; <Validate> to check all attributes have been populated correctly; or <Cancel> to exit without saving the entity.

 

 Cut or copy an entity

Processing entities can be created and / or moved within the library tree by using the cut, copy and paste function. The parent and associated children in the library tree will be moved, if the entity selected is a parent.

  1. Select the desired entity on the tree and right click. Select cut or copy, as appropriate.
  2. Select the position on the tree where the entity is to be located. Right click and select paste.
  3. If copying an entity, the new entity will open. Enter a unique code and description for the new entity. All other entity attributes will be populated with the same values as the original entity.
Note

The add, cut, copy function is not applicable to company entities. Companies are created in System Administration.

 

Modifying Entities

Edit the attributes of an entity
  1. Select the entity on the library tree to view the attributes on the right of the screen.

  2. Make any changes as required by selecting <Edit>.

  3. Select Validate to check the fields are correct before saving or cancelling.

  4. Select Save to save changes or Cancel, to exit without saving.

 

Trashing (deleting) Entities

Trash an entity

It is not possible to delete an entity in pControl, once it has been saved, as there may be processing history related to the entity. Entities created in error can instead be made inactive and moved to the 'Trash' directory.

  1. Select the entity from the tree, right click and select 'Cut'.

     

  2. Navigate to the 'Trash' directory at the end of the entity tree; right click and select 'Paste'.

    Note
    The 'Trash' directory will only be visible on the library tree if the user has access to it.
