OSP002_UPM_OS

Process Template Option (UDV): OSP002 - Category Validation for UPM_OS Category Code (CAT_UPM_OS_UDV)
Business Description

This validation checks that the difference between the actual value of the category versus the expected value of the category as a proportion of the NAV is within tolerance.
Processing Detail

$\small Absolute$ $\left [ \frac{Actual\ Value\ of\ Category\ -\ Expected\ Value\ of\ Category}{NAV} \right ] >$ $\small tolerance$
