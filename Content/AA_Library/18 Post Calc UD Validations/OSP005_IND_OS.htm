OSP005_IND_OS

 

Process Template Option (UDV): OSP005 - Validate OMAS Index Movement (CAT_IND_OS_UDV)
Business Description

This validation checks that the movement in the value of the index category is within tolerance.
Processing Detail

$\small Absolute$ $\left [\frac{(Current\ Value\ of\ Category\ -\ Prior\ Value\ of\ Category)}{Prior\ Value\ of\ Category} \right ] >\ $ $\small tolerance$
