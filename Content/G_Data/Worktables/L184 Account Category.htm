Indicates the category of the asset, e.g. OS (Ordinary Common Stock), DS (Cash Deposit), FI (Fixed Income). Each of these categories must be present in the L220: Instrument Type Mapping static worktable.

If the entity's assets can be grouped into categories, then the category code for each asset can be defined here. The asset categories can then be reported in the Analysis Views.
