U496: Classification Allocation Parameters

Classification Allocation Parameters are used in conjunction with the Classification Structure (U495). They define between 1 and 5 compliance checks to be performed on any proposed transactions generated for the linked Plan entity.

Derived values are also defined here using the 'Component' and 'Component sign' fields. Each component represents a node within the existing Classification Structure. The component sign is used to add or subtract the value from the total of these components.

Note

The U495: Classification Sectors worktable must be populated prior to populating this worktable.
