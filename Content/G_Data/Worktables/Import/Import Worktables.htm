Import

Import worktables contain data supplied from external sources, including, but not limited to, an external application, database, email, csv file.

Data is usually loaded into pControl's import worktables automatically using the Data Manager product, or other automated tool. In some cases, data can be manually entered into an import worktable, however this is dependent on a user's access levels.

 

p_Status Fields

Most worktables in pControl contain a p_status field. The p_status value indicates the status of an individual record. pControl processes use the status value to determine whether or not to select a record for inclusion in a process.
