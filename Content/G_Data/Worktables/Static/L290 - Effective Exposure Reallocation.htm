L290: Effective Exposure Reallocation

The L290: Effective Exposure Reallocation worktable lists the individual assets of the plan whose value is to be transferred from one entity to another.

Note

Reallocation is not a transfer of ownership. The title of the asset remains with the 'Source' entity, as listed in the L290: Effective Exposure Reallocation worktable.

Indicates if this row is active, or not.

Where a row is active, (check box selected), reallocation of the asset's value will occur when Cash Allocation or Rebalancing processing is run.

Where a row is inactive (check box unselected), the row will not be included in any Cash Allocation or Rebalancing processing.

 Sample Worktable Data

Processes that use this Worktable

Portfolio Management uses this worktable in conjunction with the U310: Exposure Reallocation Source to reallocate the value of the asset listed in the 'Asset Code' field to the 'Destination pControl Code entity' during the Cash Allocation process.
