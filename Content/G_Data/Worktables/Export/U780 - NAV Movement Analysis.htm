U780: NAV Movement Analysis

Holds the NAV Movement Analysis and Oversight Comparison Analysis report data for a price record. The output from the NMA and OMAS reports are also written to the U780: NAV Movement Analysis export worktable.

The standard export buffers are used by the pricing process to store or export all values that may be required for follow-on reporting, analysis or processing purposes. The information stored in export buffers cannot be updated by a user and represents a permanent record of calculated values or values that were used in a process.

The export buffers that are populated by a pricing process will be determined by the configuration of the process template Data Export processing phase. For NAV Oversight, the ‘Export NAV Movement Analysis’ attribute on the process template should be selected if data needs to be written to U780: NAV Movement Analysis.
