Data Manager

pControl Data Manager is used to facilitate inbound and outbound movement of data within pControl. It performs the task of data Extract, Transform and Load (ETL) from external or internal sources into pControl buffers for use in further internal processing or as a storage place, before being exported in various formats.

The automated features of Data Manager include:

  • Data Monitoring: Detection of data as it becomes available in defined locations, for use by pControl processes
  • Take On: Loading data from external sources into pControl from file or database
  • Data Enrichment: The preparation of import data for use by other processes
  • Data Validation: Validation of data based on specified criteria
  • Hand Off: Exporting of data from pControl to a file or another database
  • Report Pack: The generation of a grouping of ‘Hand-Off’ reports
