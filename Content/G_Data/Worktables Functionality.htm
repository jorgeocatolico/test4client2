Worktables Functionality

There are a number of actions that can be taken across multiple worktables, irrelevant of type, that are described below.

Adding, viewing and deleting attachments

Attachments are enabled through the security configuration within User Permissions. Once configured, attachments can be added, viewed or deleted, by:

  • Selecting the icon in the attachments field to view attachments; or
  • Selecting <Properties> from the ellipsis menu to open the Properties view. Select the <Attachments> tab to view attachments.

 

 

 Worktables with pending changes

When dual authorisation functionality is activated, and a change made to a worktable item that requires approval, the pending change will display under the Worktables tab of the Pending Changes Dashboard.

 Totalling and Selected Totalling

pControl worktables can be configured to display 'Grand Totalling' and 'Selected Totalling' (the sum of selected rows) as shown below.

 

Configuring Totalling and Selected Totalling (in System Administration)
Note

For further details regarding creating or maintaining data views, refer to Data Views

  1. From within System Admin, select Maintain > Data Views

    To view all existing data views.

  2. Select "Worktable / Worktable" from the data view type drop down list at the top.

  3. Select the required data view, and select <Modify>.

  4. Select the 'Table Display' button,

    and select 'Grand Totalling enabled' and 'Selected Totalling enabled' check boxes.

  5. Select to save the changes

    or

    Select to exit without saving changes.
