Understanding Processing Status

The Processing Status panel

The Processing Status panel shows a high level view of the status of a process for the entity selected in the library tree. This high level view can be expanded to show the status of the various stages that make up the process, as well as further detail about the functions and calculations performed. If a process is in a state of 'waiting' the status messages will detail what the process is waiting for.

Note

The fields displayed in the image below may change, depending on the data views available.

 Performing a Processing Status query

Using the primary filter, select the relevant data and select search to run the query.

  1. Select the relevant entity (Optional)

  2. Select the date field, in the menu bar, then select:

    • the required date range from the calendars displayed, or the list of commonly used dates
    • the 'date type' from the drop-down menu
    • Ok to save the date selections.

  3. Select the relevant process, function or data type from the menu (Optional).
  4. Select a process status from the 'All Status' drop down menu (Optional).
  5. If the query is related to a run id other than the default '001' change the run id as required (Optional).

  6. Select the Search button to run the query, and return data to the panel.
