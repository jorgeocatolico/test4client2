Fund Sign Off

Many pControl processes result in the creation of a record known as a 'Control Record'. Control records include unit price records, security price records, FX rates and imported data records. All control records are displayed in the 'Fund Sign Off' view. Additional views that can be accessed via this page, contain further detail about the processing status of that record, exception events, calculation details and others.

The Fund Sign Off view enables the monitoring of the progress of each control record in detail. The Fund Sign Off shows all of the control records that have been initialised, calculated or loaded into pControl for the current group of entities selected across the date range selected.

Some of the fields captured in the Fund Sign Off view are described in the table below. The view is maintained via Functionality Scope > Dataviews in the Web Admin module including the ability to restrict comments from different user groups, subject to user security access levels.

Field

Description

The current status of the control record in the 'Control Record - Approval' Status Stream.

The name of the entity.

Currency

The currency of the control record - this is always the base currency of the entity.

Effective Date

The date that the price is to be used for i.e. is effective for.

The type of record (price or valuation point) this represents, e.g. daily pricing.

Control Record Values

The control record values shown will depend on the processing that has been run for that entity i.e. if Valuation 1 is run on a benchmark, the control records will show an index value, if Valuation 1 is run on a currency, the control record will display an FX rate, and if unit pricing is run on an entity, the control record will display the various price types calculated (NAV, Buy, Sell and GAV).

The calculations that are completed for a particular entity are configured through the selection of functions on the process template.

Process Phases Status

The status of the processing phases associated with processing which have status streams attached (Process Validations, Data Loads, Interfund Impact etc).

 

 Contents of the Ellipsis Menu

Selecting the ellipsis on a control record gives access to further detail about the processing status of the record, exception events, calculation details, worktables and the ability to transition to the record to a new state.

Note

This Fund Overview screen is different to the Fund Overview component on the dashboard.

 

 Bulk Submit

The check box at the top of the Fund Signoff table selects all visible control records, up to 5,000 records. Where pools are expanded to display the classes underneath and bulk select is applied, the pool and its visible expanded classes will be included in the bulk select. For example, the entity named 4669_CLASS and the class listed below 4669_CLASS_B would both be included in the bulk submit selection.

Once bulk selected, comments and attachments can be added to multiple records, by scrolling to the right of the screen and selecting the comment and/or attachment icons. Transitions can be carried out across multiple control records, where a common transition is available, by selecting the ellipsis. Detailed steps for transitioning a record are listed under Transitioning to a new state on the approval status stream.

 Approval Status Streams

A status stream is a predefined flow of states and transitions that form a process. pControl uses status streams for the processing of business objects; transitions provide paths between states.

The Price Status column on the Fund Sign Off view indicates the current state of the price on the status stream. The following diagram shows status streams in practice.

An explanation of these states and their available transitions are described under Approval Status Stream Detail.

 

 Exceptions

Selecting the Exceptions option from the ellipsis menu navigates to the Exceptions view where records can be viewed and transitioned as described in Exceptions .

Note

It is not possible to clear Exceptions from this screen. Exceptions must be cleared from the Exceptions view.

 

 Worktables

Data within pControl can be viewed via worktables, also known as buffers. The worktables that are listed in the ellipsis menu are relevant to the record selected and can be viewed and/or edited in the Worktables view.

 

 Transitioning to a new state on the approval status stream

It is possible to transition the price status to the next available state by using the ellipsis menu.

  1. Select the ellipsis and choose the new state from the options available i.e. Not Released, Pending Approval, Run Again.

     

  2. In some instances, a comment will be required before the transition can be completed.

     

Note

It is not possible to clear Exceptions from this screen. Exceptions must be cleared from the Exceptions view.
