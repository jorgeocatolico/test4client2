Holiday Calendars

'Holiday Calendars' are used to define non processing days, e.g. public holidays. One or more holiday calendars can be combined with a 'Processing Calendar' using a 'Calendar Schedule' to ensure processing is not executed on non working days such as public holidays.

Add a Holiday Calendar
  1. Select the plus icon , or

  2. Enter a unique code and description for the holiday calendar:

    Or select the folder icon in the bottom left hand corner, and select the xml file...
    Select the xml file from the network location it was saved to, and select <Open>.
    The code and description fields will be automatically populated , and the xml file name will appear next to the folder icon.

    Note
    For details on exporting a calendar as an xml file, refer to Download a Holiday Calendar (as an xml file)
  3. Select <Save> to save the changes and create the calendar; or <Cancel> to exit without saving the changes.

 

 Copy a Holiday Calendar
  1. Select the icon

  2. The 'Copy Holiday Calendar' window will open, showing the source calendar being copied at the top.

    Enter a new code and description for the calendar.

  3. Select <Save> to save the new calendar; or <Cancel> to close the window without saving.

    Note
    If a new calendar is created, complete the steps under Edit a Holiday Calendar to specify non-processing dates.

 

 Edit a Holiday Calendar

  1. Select the edit icon,, to change the calendar code or description.

    Or

    Select the cog icon,to edit details of the calendar.

    The calendar detail will be displayed on the right hand side of the screen. Select the non-working dates by selecting individual dates.

    Note

    Holiday dates for future years need to be selected by navigating to future years and repeating this step.

  2. Select <Save> to save the calendar; or <Cancel> to exit without saving the changes.

  3. If the holiday calendar is being created for the first time, select <Save> to save the holiday calendar; or <Cancel> to exit without saving.
