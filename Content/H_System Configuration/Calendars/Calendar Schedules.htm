Calendar Schedules

Calendar Schedules are created as a stand alone entity type, and are a means of overlaying one or more holiday calendars with a processing calendar to specify valid processing days. Calendar schedules are then associated with a processing entity via the 'Pricing Calendar' attributes.

Example

In the calendar schedule below,

  • Green squares are derived from the DAILY_PRICING calendar, and indicate valid processing days;
  • Orange squares are derived from one of the holiday calendars.

Where one date is included in both calendars, such as 16 May, pControl uses the 'Calendar roll web' attribute to determine whether this is a valid processing day or holiday.

  • Where roll = 'Forward' or 'Backward', 16 May will be a holiday
  • Where roll = 'Never', processing will be performed on 16 May.

 

Create a Calendar Schedule Entity

  1. Navigate to Library > Entities. Select the location in entity tree where new entity is to reside.

    Right click, and select Add

  2. Select:

    • 'Funds Processing' from the Template Group drop down menu, and
    • 'Calendar Scheduling Template Web' from the Template Type drop down menu.
    Note
    'Calendar Scheduling Template' is a different entity type to 'Calendar Scheduling Template Web'. The first is only used in the processing mode.

     

  3. Select <Continue> to save the changes and continue, or <Cancel> to close the window without saving.
  4. Enter a unique entity code and description; select the 'Active' flag,

  5. Navigate to the 'Common' tab, and select the required Process Calendar, in this case the 'Daily Pricing' was selected.

    Tip

    The arrow on the entity navigates direct to the calendar on the schedules page.

  6. Scroll down and select a start date; processing frequency; and calendar roll type:

     

     

  7. Select <Save> to save the new entity; <Validate> to validate all fields have been populated correctly, or <Cancel> to exit without saving.

  8. Navigate to Configuration > Calendars.
  9. Select the 'Schedules' tab.

  10. Select the settings icon,next to the entity name. The schedule will open and the Processing calendar selected on the entity will be applied and the listed on the right of the view.

  11. Navigate to the Processing tab, on left side of screen, and select 1 x processing calendar;

    Navigate to the Holidays tab, on left side of screen, and select 1 or more holiday calendars, by selecting the plus icon, next to the required calendar.

    The schedule entity will then be display an overlay of all calendars selected:

  12. Select <Save> to save the changes, <Validate> to validate the changes made, or <Cancel> to exit without saving changes.

 

 Edit a Calendar Schedule Entity
  1. Navigate to Configuration > Calendars

  2. Select the 'settings' icon,next to the calendar schedule to be edited.

  3. The calendar schedule entity will open displaying processing and holiday dates overlayed:

  4. Delete calendars from the schedule by selecting the delete icon, on the right hand side of the screen.

  5. Add additional holiday calendars, by selecting the 'Holidays' tab, and then the add button next to the required holiday calendar.

  6. The Calendar Schedule will now show the three holiday calendars which have been added.

    Select <Save> to save the changes, <Validate> to validate the changes made, or <Cancel> to exit without saving changes.
