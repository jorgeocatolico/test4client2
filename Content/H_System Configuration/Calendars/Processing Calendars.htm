Processing Calendars

'Processing Calendars' are used to define valid processing days. They are then combined with a 'holiday calendar' on a 'Calendar Schedule' to restrict the number of valid processing days.

Note

It is not possible to associate a processing calendar directly with a fund entity. The processing calendar must first be associated with a Calendar Schedules entity, and then the calendar schedule entity is associated to the fund.

 

Add a Processing Calendar
  1. Select the plus icon , or

  2. Enter a unique code and description for the processing calendar:

    Or select the folder icon in the bottom left hand corner, and select the xml file...Select the xml file from the network location it was saved to, and select <Open>
    The code and description fields will be automatically populated , and the xml file name will appear next to the folder icon.

    Note
    For details on exporting a calendar as an xml file, refer to Download a Processing Calendar (as an xml file)
  3. Select <Save> to save the changes and create the calendar; Or <Cancel> to exit without saving the changes.

The processing calendar will now be created (if <Save> was selected above); Calendar days and dates can now be specified. Refer to 'Edit a Processing Calendar' for further details.

 Copy a Processing Calendar
  1. Select the icon

  2. The 'Copy Processing Calendar' window will open, showing the source calendar being copied at the top.

    Enter a new code and description for the calendar

  3. Select <Save> to save the new calendar; Or <Cancel> to close the window without saving.

    Note: If a new calendar is saved, complete the steps under 'Edit a Processing Calendar'.
