User Maintenance actions

The following table provides a description of each field on the User Maintenance view.

Field

Description

 

Search

 

Used to search for a user id or name. (It is not possible to search other columns)

Type the user id or user name into the search field, and the rows displayed will be filtered to the search criteria.

 

Filter

The list of users displayed can be filtered using the drop down menu to the right of the search box.

Tip

Use the search filter without entering a search term to reduce the number of users displayed.

 

 

Create New User

Selecting this button will open the 'Create User' page.

 

User ID

Displays the User ID.

 

User Name

Employee's user name is displayed.

 

Phone

User's phone number or extension number if included.

 

Email

Email address of the user if included.

 

Employee Identifier

Unique employee identifier such as an employee number.

 

Date Created

Date on which the user has been created.

 

Last Updated

Date of last edit.

 

Active

If selected, the account is active.

Select the toggle to change the account from active to inactive or vice versa.

A confirmation prompt will appear prior to selecting Activate / Deactivate or Cancel as appropriate.

This action will be confirmed with the following message:

 

Locked

If selected, the account is locked and must be unlocked to allow the user to log on.

Select the toggle to lock or unlockthe account.

A confirmation prompt will appear prior to selecting Lock / Unlock or Cancel as appropriate.

When confirmed, the following message will be displayed:

 

Duplicate

Selecting the duplicate icon will open a copy of the selected profile where settings can be viewed and edited as described below.

 

Delete

Deletes the current user.

Once selected, a message will be displayed to confirm the action. Select "Delete" to confirm or "Cancel" to exit without saving changes.

If the user id has been used to log into pControl, and is then deleted, it won't be removed from the User Maintenance tab, (as it must remain for audit purposes), however the delete and edit icons will be greyed out, and it can no longer be edited or reactivated. It is also not possible to reuse this user id.

Warning
Deleting a user cannot be undone.

 

 

Edit

Selecting the edit icon, will open the individual's profile where settings can be viewed and edited as described below.

 

 

 Create a new user
  1. Create a new user by selecting

  2. Complete the mandatory fields.

    Mandatory fields are marked with * and highlighted with a red border if left blank. It is not possible to save the new user without these fields being completed.

    • User ID
    • User Name
    • Password
    • Confirm Password.
  3. Select and a confirmation will display.

 

 View or edit a user
  1. Users can be searched for using the search box and/or the list of users can be filtered using the drop down box.

  2. From the list of user names, find the required user and select anywhere on the row.
  3. Make required edits, for example:
    • edit User Details including phone, email, employee identifiers
    • edit Access settings including activating or deactivating a user, or locking and unlocking the account
    • add or delete Roles allocated to the user.

    All fields are described in the table below:

  4. Select and a confirmation will display.
  5. Once completed, the last updated date beside the user name will be updated.

 

 Delete a user
  1. In the same line as the user name, select the delete icon
  2. A message will display to confirm the action.

  3. Select "Delete" to confirm or "Cancel" to exit without saving changes.

    Warning
    Deleting a user cannot be undone.

     

Note

Deleted users will no longer be displayed under User Maintenance.

Although the user cannot be reinstated after deletion, the record remains on the database for audit purposes.

'Deleted' user ids may not be re-used. However, if a user is set-up and never logs into pControl, they may be deleted from the database and the user id reused.

 

User Maintenance Actions within an Individual Profile

The options within an individual's profile are described below.

Add or remove user roles to a user
  1. In the same line as the user name, select the edit icon
  2. In the user profile, select the role name from the column on the right or use the search function.
  3. To view those already assigned, select the "Show allocated roles only" box and the list will filter below. A user can have multiple user roles allocated.

  4. Add or remove roles by selecting or deselecting the check box beside the User Role name.

  5. Select and a confirmation will display.
