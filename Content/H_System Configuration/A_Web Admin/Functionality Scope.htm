Functionality Scope

Configuration of Functionality Scope

The configuration of functional roles is used to allow users varying levels of access to different components of pControl.

Create a functional role
Note

Functional Roles are created in an 'inactive' state. At the completion of the wizard creation process, the role must be activated by selecting the toggle from to

  1. Copy an existing functional role by selecting the duplicate icon ; or create a new role by selecting the create button

  2. Enter a title for the functional role into the field at the top, for example, "Fund Accountant" or "Supervisor".

    3. Note
    It is possible to duplicate names, however it is not possible to complete the role creation process without a title.

  3. Select the relevant web views, and any associated panels or sections of the web view. Select

  4. A list of available data views (for each web view) will appear.

    Select the 'view', 'add', 'delete' or 'modify' column next to the required data view, to enable access. Where parent and child relationships exist, selecting the parent will auto-select all of its children. Alternatively, it is possible to select and de-select individually.

  5. Repeat the previous step for each web view activated.
  6. When all required web views and data views are selected (i.e. check boxes ticked), select

  7. The status streams related to the web and data views selected, where a manual change can be made to the data, will be displayed.

    Select the 'Enable' column check box on the relevant status stream row, to allow access to make changes to certain fields, when a record is within the state listed.

  8. The Status Stream Transitions will be displayed.

    Select the 'Enable' column check box to grant access to users within this functional role to be able to make manual status stream transitions within the pControl web application.

  9. Select

    A message will appear to confirm the save and the new functional role will appear in the list of role names.

  10. Activate the newly created functional role by selecting the grey toggle, in the 'Active' column on the Functional Scope page. The toggle will turn green, to indicate it is active.
 View or edit a functional role
  1. From the list of role names under the Functionality Scope tab, find the required role and select the edit icon . Role names are listed alphabetically, or specific roles can also be searched. Once the edit icon is selected, all relevant web and data views will be displayed and can be edited by selecting / deselecting check boxes.
  2. Complete steps 3 - 9 above.
  3. Once completed, the last updated date field will reflect the date the changes were saved.

 

 Activate or deactivate a functional role
  1. Find the appropriate functional role name from the list and select the toggle (activate) (deactivate).
  2. A confirmation prompt will appear. Select Activate/Deactivate or Cancel as appropriate.

 

 Delete a functional role
  1. Find the appropriate functional role name from the list and select the delete icon .

  2. A confirmation prompt will appear. Select Delete or Cancel as appropriate.
