Minimum Cash Total Orders for Originators

The Minimum Cash Total Orders functions in both Cash Allocation and Rebalancing processes allow a minimum order to be applied for an originator entity from a compliance check carried out pre-approval to identify whether or not the minimum is met. Where the minimum is met, the transactions will be processed. This function ensures that all orders over multiple assets total more than the Minimum Total Order for the Originator. The value of the minimum total order is the greater of the two attributes 'Minimum Total Orders ($) and Minimum Total Orders (% of investment holdings).

Where the minimum is not met, the orders will be moved to an exception state. It is possible to override or reject the proposed orders.

 

Examples

Example to demonstrate where proposed orders are more than the minimum total value order ($)

Cash at Bank

Cashflows today

Total Cashflow

$250,000

$10,000

$260,000

Minimum Cash Balance

$1,500,000

Over/under cash

$260,000-$1,500,000 = -$1,240,000 (redemption)

Note: This total is made up of two redemption orders or -100,000 from Fund A and 1,140,000 from Fund B.

Minimum Originator Orders

$200,000

Test over/under cash with Minimum Order

abs(-1,240,000)>abs(200,000) = TRUE

Therefore, orders can be placed for a redemption of $1,240,000

Example where proposed orders do not meet the minimum originator order ($) and an exception is raised

$10,000

$200,000

Therefore, the minimum originator order is not met and an exception is raised.

Example where minimum originator order is not met and an exception is raised

Cash at Bank

Cashflows today

Total Cashflow

$1,495,000

-$10,000

$1,485,000

Minimum Cash Balance

$1,500,000

Over/under cash

abs(-15,000)>abs(200,000) = FALSE

Therefore, the minimum originator order is not met and an exception is raised.

Note that despite being under the cash minimum of $1,500,000.00, a redemption is NOT raised for the minimum cash order.

Example where minimum originator order is not met and an exception is raised

Rebalancing Application

Rebalancing Redemption

Total Cashflow

$1,595,000

-$100,000

$595,000

Minimum Originator Orders

$600,000

Test over/under cash with Minimum Order

abs(595,000)>abs(600,000) = FALSE

Therefore, the minimum originator order is not met and an exception is raised.

The value of the minimum total order is the greater of the two attributes 'Minimum Total Order ($) and Minimum Total Orders (%).

Note

Where using the 'Minimum Total Orders (%)' attribute, the value is determined by multiplying the percentage value by the investment holdings value.
