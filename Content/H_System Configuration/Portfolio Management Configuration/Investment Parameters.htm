Investment Parameters

Investment Parameters work in conjunction with classification structures and are used to apply checks against target weights, and their tolerances and bounds on any proposed transaction. It is possible to configure investment parameters to a single originator plan and a multi-level plan.

The Investment Plans panel displays all plans according to the search criteria selected in the filters. Selecting a plan, will open the second panel which displays all investment parameters associated with the plan.

Understanding Investment Parameters

Investment Parameters panel
Example

A Portfolio Manager may decide to make changes to Schroders HKD Infrastructure, one of the assets within the Plan H1 shown below. Any changes made are required to be validated before being committed to the system. Should the changes not fall within the target weight and its associated tolerances and bounds, a hard or soft warning will display indicating each line and its error that requires to be corrected before being validated and committed. It is not possible to commit without passing the validation stage.

 Investment Parameters example
Example

Every allocation parameter has an originator, a counterparty and an asset as well as being able to attach a plan. Plans can either be single or multiple originator. The investment structure target weight will always add to 100%, irrelevant of how many levels.

For example, a Single Originator Plan, the Australian Shares Plan invests 30% into Australian Equities, 20% into International Fixed Income, 5% into Domestic Property Sector and 45% into Global Equities, as shown below. The total target weight of the shares amounts to 100%. In the Investment Parameters panel, the originator will default to the plan name i.e. Australian Shares Plan.

For a multiple originator/multi level plan, the counterparties from the single originator plan now become the originators for the next level. For example, the Australian Shares Plan invests 30% into Australian Equities. Australian Equities then invests into Dollar and Sense AU Equities, Conviction AU Equities and MG Aus-shares. Australian Equities acts as both a counterparty and an originator in multi level plans. The total target weight of the shares across each level will amount to 100% and parameters are set which will apply checks to ensure these investments remain within their bounds and tolerances of the target weights.

In the Investment Parameters panel, it is possible to modify the originator so as to represent a multi-level plan.

 

Understanding Investment Parameters with Dual Authorisation

Using dual authorisation

 

 

Note

If Dual Authorisation is not active, the Authorisation Status column will not be displayed. Authorisation Status transitions will not be available via the ellipsis menu however, the ellipsis will be displayed in red to indicate pending changes. Dual authorisation can be activated via system administration.

 

Modifying Investment Parameters with Dual Authorisation

To validate a record

Records can be validated at any time, with or without changes by selecting <Validate> on the Investment Parameters.

A confirmation of the validated record or any hard or soft warnings will display to the right of the Investment Parameters panel.
