Linking Proposed Orders on Interfund Structures using External ID

An External ID can be used in order to link proposed orders within interfund structures. The diagram below shows an interfund structure and how it is linked using the External ID. All ALRs within the structure have their own unique ALR ID, but they all share the same External ID that has been provided for the initial cashflow via the U110: Cashflows and Orders worktable. The External ID can be input or imported via the U110: Cashflows and Orders import workable and is available in both pControl Processing Mode and pControl Web Mode.

If an External ID is not provided via the U110: Cashflows and Orders, pControl will generate a unique ID once the CAR process is run.
