Transaction Display

The Cash Forecast table displays the summed total of cash financial events for the selected date range.

The individual transactions that make up a cash forecast value can be displayed by selecting one cell in the cash forecast table, and the associated transactions will be displayed below.

The transaction display changes according to which cell you select.

When you select a cell, the corresponding transaction for that column date will be displayed on the panel below.
