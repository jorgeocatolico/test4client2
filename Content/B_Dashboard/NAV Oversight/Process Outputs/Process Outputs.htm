Process Outputs

Once the NAV Oversight process has been started, a control record is generated and displayed on the Fund Sign Off view. This view displays:

  • the current status of the process, over the life cycle of the process;
  • the status of each phase within the process, (Data Loads, Pre-Pricing Validation, Post-Pricing Validations)
  • as well as the values output at the end of the end of the process, i.e. an Expected Price, or an Actual Price.

Selecting the ellipsis on a control record gives access to further detail about the processing status of the record, exception events, calculation details, worktables and the ability to transition to the record to a new state:

  • Exceptions (if any) encountered during the process
  • Fund Overview which shows:
    • details about each phase of the process,
    • the status of each phase,
    • the values used as inputs,
    • status messages listing any calculations performed,
    • and data exports.
  • Analysis Views including Share Class Impacts, DYAS, OMAS, AMAS, and NAV Movement Analysis reports.
