NAV Oversight

NAV Oversight is the term given to the validation of the components of the NAV. It provides details on the components of prices that an organisation is aiming to check, including the following:

  • Calculation of an expected price for portfolios or funds, to be used in the validation of prices struck by service providers
  • Validation of actual price data provided by the service provider
  • Generation of a detailed movement analysis report for a price; to explain its movement from the previous pricing point
  • Generation of a detailed oversight analysis report for a price; to compare its movement from the expected price
  • Validation of the movements in the various components of a price between the expected and actual prices.

The pControl NAV Oversight product provides Investment Managers with automated, unified oversight of all outsourced service provisions. The NAV Oversight solution offers a single point of reference to monitor, validate and analyse data from service providers in a timely and accurate fashion. It provides Fund Managers with an effective means of control, ensuring service level agreements (SLAs), are met and service to their clients is maintained.

As well as aggregating, validating and presenting fund related data to key business stakeholders, this product can also facilitate 'sanity checks' across the business, and drill down into transaction and fund data where further examination or validation is required. The existing movement Analysis Views are tailored to give specific information on all aspects of the outsourced provision.

 Oversight Pricing

The Oversight pricing process is sometimes referred to as the 'Daily Pricing' or 'Unit Pricing' processes, as it is usually run under an entity's 'Daily Pricing' tab, and uses many of the same calculation and validation functions as the Daily Pricing process. This process also produces an overall price. This price, as well as the components or categories of the price are detailed in the Oversight Comparison Analysis and NAV Movement Analysis  reports for comparison and investigation purposes.

 

 Expected Pricing

The expected pricing process calculates an ‘expected GAV’ and 'expected NAV', by taking the last known Actual NAV and its constituents (Fixed Income, Capital Impacts, Transactions, Fees, Cross Holdings etc), from the last approved Oversight process (Daily Pricing process), and calculating the expected movement for each constituent.

 

 Backup NAV Pricing

Backup NAV pricing uses the same process template type as the Expected Pricing process, so it is possible to use the same template for each process. Alternatively, different templates can be configured for each process, to allow some differences in functionality between the two processes.

Backup Pricing process is only run when a service provider declares an 'outage' and is therefore, unable to supply the NAV prices. When this occurs, Backup NAV pricing can be run each day, until such time as the service provider is able to resume normal service provisions.

 NAV Oversight Process Entities

NAV Oversight can be run on Oversight Pool, Oversight Class or NAV Oversight entities. There are also several other entity types used to support the processes as listed in the table below.

Icon Entity Type Description
Expected Pricing Class Process Template

This entity contains all available functions of the Expected Pricing process for Oversight classes. Functions are turned on or off by selecting / deselecting the check boxes.

This template is also used for Backup NAV processing.
Expected Pricing Pool Process Template

This entity contains all available functions of the Expected Pricing process for Oversight pools. Functions are turned on or off by selecting / deselecting the check boxes.

This template is also used for Backup NAV processing.
Expected Pricing Process Template

This entity contains all available functions of the Expected Pricing process for other entity types. Functions are turned on or off by selecting / deselecting the check boxes.

This template is also used for Backup NAV processing.
Oversight Class Process Template

Process template containing all available functions that can be executed during the actual pricing / oversight pricing process.
Oversight Pool Process Template

Process template containing all available functions that can be executed during the actual pricing / oversight pricing process.
