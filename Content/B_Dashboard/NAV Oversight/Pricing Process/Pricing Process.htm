Pricing Process

Expected Pricing and Oversight Processing processes are configured via 'Process Templates', which are associated to Oversight entities within the Processing Module. Each template is made up of functions which are grouped into processing phases.

 

The following process flow diagram shows the inputs to the NAV Oversight process; the processing phases with a selection of functions; and the process outputs. This is only a sample. There can be many more worktables used, and many more process functions turned on. For further detail on the most commonly used functions, refer to the NAV Process Phases and Functions - new page.
