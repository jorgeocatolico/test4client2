NAV Oversight Dashboard

NAV Oversight is the term given to the validation of the components of the NAV. It provides details on the components of prices that an organisation is aiming to check, including the following:

  • Calculation of an expected price for portfolios or funds, to be used in the validation of prices struck by service providers
  • Validation of actual price data provided by the service provider
  • Generation of a detailed movement analysis report for a price; to explain its movement from the previous pricing point
  • Generation of a detailed oversight analysis report for a price; to compare its movement from the expected price
  • Validation of the movements in the various components of a price between the expected and actual prices.

Understanding the NAV Oversight Dashboard

NAV Oversight Dashboard

The NAV Oversight Dashboard consists of five widgets, or panels, that can be configured in System Administration. Status streams underpin all of the components in the dashboard and are described in detail under Exceptions Status Stream states and transitions and Approval Status Streams.

 Outages Shortcut bar

The shortcut bar on the left side of the dashboard contains icons for quick navigation to various screens, or for populating the primary filter.

Library Tree will display all entities in the tree. The entity selected here will populate the primary filter at the top of the dashboard, for filtering the data displayed.

Note: To collapse the Library Tree, select the arrow next to the search field, or select the Library Tree icon again.

Selecting the Worktable icon, opens the Worktables menu

Selecting the 'Default' icon from any web page navigates back to the default page, which will usually be the Dashboard.

Additional icons can also be added to the shortcuts menu bar. These are configured in the 'Web Global Defaults' configuration lookup within System Administration module
