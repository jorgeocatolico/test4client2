AMAS Specification Table

Standard categories are driven by balances from the U104: General Ledger with calculations specified in the table below.

Categories not populated by the U104: General ledger will be populated as follows:

Indices (IND_AMAS)

  • IND1_AMAS and IND2_AMAS will be populated with values of Key Benchmark and Dynamic Benchmark respectively, as defined on the entity.

Unit Price (UPR_AMAS)

  • Populated with the NAV Price from the U101: External Prices worktable.

Net Asset Value (NAV_AMAS)

  • Populated with the Net Asset Value from A01: Product Validation workable.

Tolerance Value

Where CAT011 has been performed on the category, the tolerance value is populated with the CAT011 tolerance value from the L200: Tolerance worktable.

Where CAT011 has not been performed, this field will left blank.

Tolerance Level

Where CAT011 has been performed on the category, the tolerance level generated by the validation, for example, Exception.

Where CAT011 has not been performed, this field will be set to 'Not Checked'.
