DYAS Daily Backward Distribution Example

The following example for the Milestone Group Fund, MGFUND_DYAS_01, shows the distribution yield analysis report for a 'Daily Backward' distribution frequency.

Differences in Daily Forward and Daily Backward calculations are included in DYAS Specification Tables.

Worktables

Load import worktables

The following worktables may be required.

  • U104: General Ledger (shown below)
  • U108: Units on Issue
  • U101: External Unit Prices
  • U350: Classification Hierarchy

 

 Load files for the static worktables:
  • L184: Account Category (shown below)
  • L315: Account Code Mapping
  • L200: Tolerance

 

Functions and Attributes

Select the required attributes and functions on the entity or process template:
  • [DL190] - Load General Ledgers
  • [DL120] - Load External Unit Prices
  • [DL070] - Load Units on Issue Balances
  • [VA860] - Generate Distribution Yield Analysis Data (shown below)
  • [CAT015] - Yield Impact Check
  • [EX093] - Export Distribution Yield Attribution Data

Exception Events

Review the Exceptions Dashboard on Monday 19th November

A [CAT015] Yield Impact Check will raise an exception for categories that have exceeded the specified tolerance in the L200: Tolerance worktable. These exceptions require review before transitioning. Selecting the row will display some further detail on the exception.

 

The DYAS View

Navigate to the DYAS

Navigate to the DYAS view using the ellipsis and selecting Analysis.

 

 The DYAS view/report and the U783: Distribution Yield Analysis Data worktable

The information in the DYAS view/report is used to analyse the yield impact. Selecting the Related Funds tab displays other funds that have encountered the same exception for this asset.
