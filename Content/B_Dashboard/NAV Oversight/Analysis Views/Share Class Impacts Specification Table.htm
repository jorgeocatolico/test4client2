Share Class Impacts Specification Table

The formulas shown below for 'Class Impact (bp)' apply to both the 'Master Class Impact (bp)' and 'Actual Class Impact (bp)' columns.

The 'Difference (bp)' column is not shown in the table below, however this is calculated as:

$\small Master\ Class\ Impact\ -\ Actual\ Class\ Impact$

The results of the validation checks and are compared to the L200: Tolerance values, and the tolerance status is displayed in CPAS analysis view.

Category Code

Category Name

Class Impact (bp)

UPI_CP

NAV price return

$\frac{U101\ nav\ price}{U101\ prev\ nav\ price - 1} \times 10000$

CUPI_CP

Clean Price Return

$\small NAV\ Price\ Return\ (UPI\_CP) - Performance\ Impacts\ (IMP\_CP)$

IMP_CP

Performance Impacts

Sum of all underlying performance impacts

DIV_CP

Dividend Impacts

$\frac{U101\ gross\ dist\ paid}{U101\ prev\ net\ nav\ amount} \times 10000$

FEE_CP

Fees

$\frac{Change\ in\ Fee\ Accrual\ Account\ Since\ Last\ Valuation\ Point}{Previous\ Net\ NAV\ Amount} $

FX_CP

FX Impact

$\left ( \frac{Current\ FX\ Rate}{Previous\ FX\ Rate} \right ) -1$

HDG_CP

Hedging Impact

$\small U101.hedge \_\ ratio\ \times FX \_\ CP\ \times -1$

RND_CP

Rounding Impact

$\small Rounding\ Impact\ = Unrounded\ NAV\ price\ return\ - Rounded\ NAV\ price\ return$

$\small Unrounded\ NAV\ Price\ Return\ = \normalsize \frac{U101\ Unrounded\ nav\ price}{U101\ prev\ unrounded\ nav\ price\ -1}$

$\small Rounded\ NAV\ Price\ Return\ = \normalsize \frac{U101\ nav\ price}{U101\ prev\ nav\ price\ -1}$

RNDCAP_CP

Capital Rounding Impact

$\small Rounded\ Unit\ Change\ = \normalsize \frac{Capital\ Flows}{Prev\ Rounded\ NAV\ price}$

$\small Unrounded\ Unit\ Change\ = \normalsize \frac{Capital\ Flows}{Prev\ Unrounded\ NAV\ price}$

$\small Value\ Difference\ = \left ( Unit\ change\ -\ Unrounded\ Unit\ Change \right ) \times Previous\ Rounded\ NAV\ Price$

$\small Impact\ = \normalsize \frac{Value\ Difference}{Previous\ Net\ NAV\ Amount}$

FWDCAP_CP

Capital Forward Impact

Client specific. This row will display zero, unless otherwise configured.

DILCAP_CP

Capital Dilution Impacts

 $\large \frac{\left ( \frac{\left ( Previous\ NAV\ Price\ -\ Current\ NAV\ Price \right )\ x\ Total\ Cashflows} {Previous\ NAV\ Price} \right ) } {\small Previous\ Net\ NAV\ Amount\ x\ 10000}$

All

Account Impact

$\frac{\left ( sum\left ( U104.Base\ Amount \right ) -\ sum\left ( Prev\ U104.Base\ Amount \right ) \right )} {Prev\ U101.NAV}$

Note: This price impact is used to calculate the impact based on movement in account balances for an account group. It can be used for any price type.

The U104 accounts must be mapped to an account group in the L184: Account Category and and L315:Account Code Mapping worktables prior to generating the CPAS analysis view.
