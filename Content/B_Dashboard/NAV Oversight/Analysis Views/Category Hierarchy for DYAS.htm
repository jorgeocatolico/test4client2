Category Hierarchy for DYAS

The Category Hierarchy used by the DYAS report is configured in the U350: Category Hierarchy worktable, and is used to order the categories within the DYAS view, as shown below. Prior to use with NAV processes, the category hierarchy must be defined in the worktable to allow the movement in distribution yield impact to be categorised according to internal classifications. A number of standard categories have been defined in the U350: Classification Hierarchy worktable for use in the DYAS report and can be recognised with the '_DYAS' suffix.

The following table shows the standard DYAS category hierarchy. These values can be amended, or added to, depending on a user's access, either by manually updating, or uploading a file to the U350: Category Hierarchy worktable.
