Analysis Views

Price calculations and validations are performed on NAV Oversight and Pricing entities during NAV Oversight and Backup NAV processing. The data and calculations used for each price are presented, according to category types, on the following analysis views:

  • The NAV Movement Analysis (NMA) view explains the price movement of an entity from the previous pricing point
  • The Oversight Comparison Analysis (OMAS) view compares and explains the price component differences between the NAV Oversight process and actual pricing process
  • The Share Class Impacts (CPAS) provides a view across classes within the same fund to ensure their returns are moving in line with each other in an expected and predictable way.
  • The Account Movement Attribution (AMAS) screen displays the movement from previous pricing point in the components of the NAV, based on the General Ledger (GL) account balances. The report is presented in both Balance Sheet and Income Statement format.
  • The Distribution Yield Analysis (DYAS) report displays a day over day or period over period view into the components and impacts a Pool or Class' distribution yield in a framework that is similar to the NAV Movement Analysis.

U350: Category Hierarchy in pControl Processing Mode Exporting and Printing the Analysis views

All of the analysis views can be exported from pControl and printed via the following steps:

  1. From the All Prices (Ctrl+P) screen, select the price record that you would like to print the NMA / OMAS for.
  2. From the menu select the icon. The following will be shown:

  3. From the options select the tab that you wish to print e.g. <NAV Movement Analysis> or <Oversight Comparison Analysis> etc. or select <Cancel> to exit the screen.

  4. From the window displayed, select or enter the required information:

    • Look in – the directory that the file is to be saved in
    • File Name – Name of file to be saved
    • Files of Type – Select type of output (i.e. XLS, CSV, PDF, HTML or XML).

  5. Select <Save> to generate the file, or <Cancel> to exit without saving the file.

  6. The following prompt appears. Once the file is opened, it can be printed.

Note

Regardless of the option chosen, the file is available in the chosen directory for future use.

If ‘Export as hierarchy’ is selected, the exported data will retain the hierarchical (indented) structure (relevant only to NAV Movement Analysis and Oversight Comparison Analysis reports).
