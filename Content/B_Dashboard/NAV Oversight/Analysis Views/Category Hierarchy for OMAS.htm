Category Hierarchy for OMAS

The Category Hierarchy used by the OMAS report, as shown below, is configured in the U350: Category Hierarchy worktable, and is used to order the categories within the OMAS report.

Note

For further details on setting up the category hierarchy and running the category hierarchy update process, refer to the NMA User Guide.
