Investment Summary

The Investment Summary displays one summary record for each plan for each day that has been processed. The initial process is generated with a batch id of 1. If the process is requested subsequent times for the same entity on a , the batch id is incremented by one. Results are shown separately, i.e. each one has a separate batch number. One summary record summarises all proposed transactions for a particular plan and effective date. These are displayed in the third panel of this view.

Understanding the Investment Summary

Investment Summary panel

The fields of this panel vary, depending on the data view selected. The coloured bar above the Investment Summary panel indicates the total number of summary records in each state of the 'Master Proposed Transaction status stream. Selecting one of these coloured sections, will filter the summary records displayed. Selecting an individual summary record in this panel will display the corresponding classifications and proposed transactions in the following two panels.

 

Modifying Investment Summary Records

To transition a record

Investment Summary rows travel along status streams, or life cycles. Most transitions between states are completed automatically by pControl. When certain states are reached, such as 'Exception', it is possible to manually transition the record to the next state.

One or many Investment Summary records can be selected for simultaneous transition provided they are in the same state and are being transitioned to the same state.

 

To transition Investment Summary rows:

  1. Select the required row or select the check boxes if there are multiple records to select.

  2. Right click on one of the highlighted rows, and the available transitions will be displayed.

  3. Select the required state from the list of available options. All of the selected rows will be updated.
