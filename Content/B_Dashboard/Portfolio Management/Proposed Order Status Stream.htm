Master and Proposed Order Status Streams

As the cash allocation process progresses through the phases of processing, there are many 'paths' that processing can take depending on the configuration of the entity and pControl static data, the data used in processing etc. The 'paths' in pControl are determined by status streams, status streams represent a predefined map of states and transitions between states.

There are two status streams, the Master Proposed Order Status Stream and the Proposed Order Status Stream which are described diagrammatically below.
