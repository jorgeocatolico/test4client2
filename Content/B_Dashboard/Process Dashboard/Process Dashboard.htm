Process Dashboard

The Process Dashboard contains several components displaying funds in different ways to enable monitoring and prioritising of processes and funds:

The Analytics Bar displays the status stream progress of each part of the process.

The Fund Overview which is associated to the price run types, i.e. daily pricing; and may also display categories of exceptions that have occurred.

The Exceptions Overview displays the number of exceptions by severity and state.

The SLA Monitor is a view of the status of the process in relation to the current time.

The Process Summary shows the progress according to each of the steps within the process.

Understanding the Process Dashboard

Process Dashboard

The Process Dashboard is used to validate fund prices and consists of five widgets, or components, that are configured in System Administration. Status streams underpin all of the components in the dashboard and are described in detail under Exceptions Status Stream states and transitions and Approval Status Streams.

 Shortcut bar

The shortcut bar on the left side of the dashboard contains icons for quick navigation to various screens, or for populating the primary filter.

Library Tree will display all entities in the tree. The entity selected here will populate the primary filter at the top of the dashboard, for filtering the data displayed.

Note: To collapse the Library Tree, select the arrow next to the search field, or select the Library Tree icon again.

Selecting the Worktable icon, opens the Worktables menu

Selecting the 'Default' icon from any web page navigates back to the default page, which will usually be the Dashboard.
Note:
Additional icons can also be added to the shortcuts menu bar. These are configured in the 'Web Global Defaults' configuration lookup within System Administration module
